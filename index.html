TipX Chrome Extension

Empower Twitter users to support creators directly via tips on the XRP Ledger.

Download Extension

Installation Instructions

1. Download the TipX Chrome Extension from the provided link.

2. Open Chrome and navigate to chrome://extensions/

3. Enable Developer Mode by toggling the switch in the top right corner.

4. Click 'Load unpacked' and select the downloaded extension folder.

5. Enjoy tipping your favorite creators on Twitter!
